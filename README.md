# CollegeApp

Static website for Gramonnati Mandal’s College of Livestock Management & Dairy Production.

## How to run
- Open [index.html](index.html) in a browser.
- For best results, use a local server (optional):
	- In VS Code, install a simple live server extension and run “Open with Live Server”.
